# airline-delay-analysis

Airline Delay Cause Analysis
Project Overview

This project analyzes airline delay data to identify the major causes of flight delays across airports and airlines. The objective is to uncover patterns in carrier delays, weather delays, NAS delays, security delays, and late aircraft delays using SQL, Python, and Power BI.

Business Problem
Flight delays affect passenger satisfaction, airline operations, and airport efficiency. This project aims to answer:

Which airlines experience the most delays?
What are the major causes of delays?
How does weather impact flight performance?
Which airports have the highest delay rates?
How have delays changed over time?
